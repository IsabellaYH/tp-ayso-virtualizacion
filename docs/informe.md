# Informe del Trabajo Práctico de Virtualización

## Introducción
Este trabajo práctico presenta una aplicación de consola en Python 3 para gestión de tareas. El objetivo principal es ejecutar la misma aplicación en máquinas virtuales clonadas para demostrar conceptos de virtualización en VirtualBox.

## Objetivo académico
- Aplicar modularización en Python.
- Validar entradas del usuario y manejo de errores.
- Ejecutar y comparar comportamiento en clones de VM.

## Arquitectura de la aplicación
- `src/main.py`: menú interactivo y flujo principal.
- `src/gestor_tareas.py`: operaciones CRUD básicas sobre tareas.
- `src/validaciones.py`: validaciones reutilizables.

## Conclusión
La aplicación permite validar de forma simple el comportamiento aislado en diferentes máquinas virtuales, manteniendo el foco en la práctica de virtualización.
